# Rimac Frontend Challenge

## Notas

La resolución en la que se verá el proyecto lo más parecido posible al diseño es 1366x768 que es en la que me basé debido al diseño de figma.

Utilicé Next.js que es una de las formas que React recomienda iniciar nuevos proyectos. [Ref](https://react.dev/learn/start-a-new-react-project)

## Cómo utilizar

Ejecute [`install`] con [npm](https://www.npmjs.com/) o [Yarn](https://classic.yarnpkg.com/en/), para instalar el proyecto:

```bash
npm install
```

```bash
yarn install
```

Ejecute [`run dev`] con [npm](https://www.npmjs.com/) o [Yarn](https://classic.yarnpkg.com/en/), para inicializar el proyecto:

```bash
npm run dev
```

```bash
yarn dev
```

Ejecute la versión de producción con:

```bash
npm run build
npm run start
# o
yarn build
yarn start
```
